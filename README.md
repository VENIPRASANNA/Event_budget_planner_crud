# 🎉 Event Budget Planner

A simple Java-based CRUD console application developed to manage event planning and budgeting efficiently.

## 📌 Project Overview

The **Event Budget Planner** helps users organize event details and manage budgets through a menu-driven console interface.

This project was built using **Core Java** concepts and demonstrates CRUD (Create, Read, Update, Delete) operations.

---

## ✨ Features

- ➕ Add New Events
- 📋 View All Events
- ✏️ Update Event Budget
- ❌ Delete Events
- 🖥 Menu-Driven Interface
- 💰 Budget Management
